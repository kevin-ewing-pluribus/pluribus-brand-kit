Pluribus Brand Guidelines

Modern, minimal, professional identity standard for anything from Pluribus.

Version 1.2.0 • aligned to blog.kevin-ewing.com logo colors

1) Core Identity

Voice: direct, clear, calm, professional.

Visual signal: clean layout, restrained accents, translucent white surfaces.

Default rule: use this system unless Kevin explicitly requests otherwise.

2) Logo + Favicon

4) Type + Layout

Typeface: Inter/system sans.

Hierarchy: H1 32, H2 22, H3 15, Body 15, Meta 12.

Spacing: 4/8/12/16/24/32/48 scale only.

Corners: 10/14/20 (soft, modern).
